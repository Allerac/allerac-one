# Allerac One — Site Map

Allerac is a personal AI assistant platform. After logging in, you access a set of specialized
workspaces (domains) through a taskbar at the bottom of the screen. Each workspace has its own
AI assistant and tools tailored to that topic.

---

## Login

- Enter username and password to access the platform
- Forgot password → receive a reset link by email → set a new password

---

## Hub (Home Screen)

The starting point after login. Shows all available workspaces as icons.

- Click any workspace icon to open it
- First-time users see a guided tour of the platform

---

## Taskbar (always visible at the bottom)

The global navigation bar, present on every screen.

- **Start menu** — open any workspace from a list
- **Open workspace chips** — switch between workspaces you have open
- **Language selector** — switch the interface language (English, Portuguese, Spanish, Catalan)
- **Dark / Light mode toggle**
- **User menu** — access your account settings or log out
- **Configuration** — open system settings
- **Clock** — shows the current time

---

## Workspaces

### Chat
General-purpose AI assistant.

- Start a new conversation
- Browse past conversations in the sidebar
- Attach files or images to a message
- Save important information to memory for future conversations
- Upload documents the AI can reference

---

### Code
AI assistant focused on programming.

- Ask coding questions and get explanations
- Paste or upload code for review or debugging
- Browse project files from the workspace
- Start and stop code execution

---

### Notes
Personal knowledge base.

- Create and edit notes in Markdown
- Preview the formatted note alongside the editor
- Search through all your notes
- Ask the AI to summarize, expand, or rewrite a note

---

### Jobs
Scheduled tasks and automation.

- View all scheduled jobs and their next run time
- Create a new automated task with a schedule
- Edit or delete existing jobs
- See the output log of past runs

---

### Finance
Financial assistant and market data.

- Ask the AI for financial advice or explanations
- Look up stock prices and view price charts

---

### Health
Health tracking and activity overview.

- View today's activity summary (steps, calories, heart rate)
- See recent workouts and activities
- Ask the AI questions about your health data
- Connect a wearable device (Garmin, Oura) in settings

---

### Email
Email inbox with AI assistance.

- Read and browse your inbox
- Reply to emails with AI-suggested drafts
- Compose new emails
- Connect an email account

---

### Search
Web search and research assistant.

- Search the web and get summarized results
- Ask follow-up questions based on search results
- Save research findings to memory

---

### Design
UI/UX and design assistant.

- Ask for design feedback and suggestions
- Get help with color palettes, typography, and layout
- Reference design system guidelines

---

### Social
Instagram management.

- Read and reply to Instagram direct messages
- Create and publish Instagram posts
- Edit images before posting
- Generate AI captions for photos

---

### Recipes
Chef and nutrition assistant.

- Ask for recipe ideas based on ingredients or dietary needs
- Get nutritional information
- Save favourite recipes in conversation history

---

### Tickets
Issue and task tracker.

- View open tickets and issues
- Create new tickets with AI-assisted descriptions
- Link issues to GitHub

---

### Content (Write)
Content creation assistant.

- Write blog posts, articles, and social copy
- Set the tone and style for generated content
- Iterate on drafts through conversation

---

### Workspace
File browser and code environment.

- Browse project folders and files
- Open and read files with syntax highlighting
- Run commands and see their output
- Kill a running process

---

### Admin *(admin users only)*
System management for the platform owner.

- Create and manage user accounts
- Assign which workspaces each user can access
- Add or remove API keys and integrations
- Connect Instagram accounts
- Configure the Telegram bot
- View system health and logs

---

## Global Features (available in every workspace)

- **Conversation history** — every workspace keeps a sidebar with past conversations you can return to
- **Memory** — the AI remembers important things across conversations; you can view, add, or delete memories
- **Documents** — upload files the AI can read and reference in its answers
- **Model selector** — choose which AI model to use for that conversation
- **My Allerac** — access your personal profile and account settings

---

## Settings & Configuration

Accessible from the taskbar at any time.

- Change interface language and timezone
- Toggle dark or light mode
- Manage connected accounts (email, Instagram, Telegram, health devices)
- Add or update API keys for external services
- View and manage your saved memories
